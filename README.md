# Proyecto-14
Predicción de cancelación de clientes y segmentación en gimnasio
En este proyecto trabajé como analista de datos para la cadena de gimnasios Model Fitness, con el objetivo de reducir la pérdida de clientes mediante el análisis de comportamiento y la construcción de modelos predictivos.

El proyecto comenzó con un análisis exploratorio de datos (EDA), donde se estudiaron variables demográficas, patrones de uso del gimnasio y métricas de comportamiento de los clientes. Se compararon usuarios activos y aquellos que cancelaron su membresía, identificando diferencias significativas en variables como frecuencia de visitas, duración de la membresía y gasto en servicios adicionales.

Posteriormente, se desarrolló un modelo de clasificación binaria para predecir la probabilidad de cancelación de clientes en el mes siguiente. Se implementaron dos enfoques de machine learning: regresión logística y bosque aleatorio, evaluando su desempeño mediante métricas como exactitud, precisión y recall, lo que permitió seleccionar el modelo más efectivo para el problema.

En una segunda etapa, se realizó una segmentación de clientes utilizando técnicas de clustering (K-means), complementadas con análisis jerárquico mediante dendrogramas. Esto permitió identificar grupos de usuarios con comportamientos similares y diferentes niveles de riesgo de cancelación.

A partir del análisis de clústeres, se evaluaron características promedio, distribuciones de comportamiento y tasas de cancelación por segmento, lo que facilitó la identificación de perfiles de clientes más propensos a abandonar el servicio frente a aquellos más leales.

Finalmente, se elaboraron recomendaciones estratégicas orientadas a la retención de clientes, incluyendo acciones específicas de marketing y engagement basadas en los segmentos identificados.

Este proyecto demuestra habilidades en análisis exploratorio de datos, machine learning supervisado, clustering, evaluación de modelos predictivos y generación de insights accionables para la toma de decisiones en estrategias de retención de clientes.
